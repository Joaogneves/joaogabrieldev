Olá! Eu sou o

João Gabriel

Apaixonado por desenvolvimento web desde que conheci, em 2021 eu tomei uma decisão, cansado de como estava minha vida resolvi começar estudar programação por conta própia, desde então não parei.

Comecei algumas linguagens até conhecer Java, até então a linguagem que eu nunca imaginaria gostar, foi a que me prendeu. Foi algum tempo estudando, aprendendo Orientação a Objetos, estrutura de dados e o Framework Spring Boot, e mesmo com várias dificuldades pela sua complexidade não desisti.

Senti nescessidade de testar minhas aplicações com um front-end, então comecei a estudar Angular, assim conseguia criar as páginas para usar o meu back-end desenvolvido em Java.

Na metade de 2023, me surgiu uma oportunidade de um curso, para uma empresa de minha cidade, curso que faço atualmemte, Desenvolvimento web com .Net framework, Angular e Sql server. Onde Aprendi C# e gostei muito da linguagem por C# e Java serem muito parecidas
